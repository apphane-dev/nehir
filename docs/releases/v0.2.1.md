# Nehir 0.2.1

## Changed

- Improved trackpad gesture handling with animation-state awareness, velocity adjustments, and more stable cumulative threshold behavior.

## Fixed

- Fixed an issue where Nehir could interfere with macOS administrator authentication sheets/dialogs, preventing Touch ID from being used in System Settings, closing [#1](https://github.com/Guria/nehir/issues/1).
- Improved IPC and CLI runtime stability by closing the server after recorded events are observed.
- Fixed workspace bar button click targets so the full intended button area is clickable.

## Internal

- Improved CI reliability by using macOS 26 runners, disabling parallel test execution, setting a timeout, and reducing warning noise.

## Notes

Nehir is currently distributed as an unsigned app. macOS may show Gatekeeper warnings until Developer ID signing and notarization are configured.

Nehir requires Accessibility permission to manage windows.
